# Interview Question Predictor (ML)

This project is a web-based application that predicts interview questions based on user input using a machine learning model.

The idea behind this project is to help users prepare better for interviews by giving them relevant questions based on their profile or input.

---

## Features

- Predicts interview questions using ML model  
- Simple and clean user interface  
- Takes user input (text/resume-based)  
- Displays predicted questions instantly  

---

## Tech Stack

- Python  
- Flask  
- Machine Learning  
- HTML, CSS  
- JavaScript  

---

## How it works

The application uses a trained machine learning model to analyze the input provided by the user.

- User enters input (skills / resume text)
- The input is processed
- The ML model predicts relevant interview questions
- Results are displayed on the screen

---

## Project Structure

- `app.py` – main Flask application  
- `ml/` – contains dataset and model  
- `templates/` – HTML files  
- `static/` – CSS and assets  
- `requirements.txt` – dependencies
